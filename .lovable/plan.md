
# PeaceCode — The Sanctuary Redesign

Cut the dashboard from 20+ cards to 8 unforgettable sections. Each one owns real estate, has one job, and feels like a room in a home — not a tile in a grid.

## The 8 Sections (top → bottom)

1. **Hero · Personal Brief**
   Full-viewport-height opening. Time-aware greeting ("Good evening, Jai"), one AI-written line of insight referencing today's real data (sleep, mood, streak), and a single primary action. No stats rail, no ring — just breath, type, and one soft CTA. Left: editorial copy. Right: a large living illustration (organic SVG bloom that reacts to mood).

2. **Today's Focus**
   One cinematic, oversized experience card (~70vh). Large image, serif title, one-line "why this, now" explanation generated by AI. Tap → begins. Previously-stacked deck collapses into a subtle horizontal ribbon of 4 alternates below.

3. **Peace AI · Ongoing Conversation**
   Full-width chat surface, not a preview. Persistent thread with Jai — remembers previous messages this session. Composer at the bottom, message list flowing above, suggestion chips only when empty. Feels like Messages, not a widget.

4. **Mood Garden**
   Replace mood bars/rings with a living illustration: an SVG garden where each day's mood is a plant (calm=fern, joy=bloom, low=bud, anxious=coiled sprig). Seven days = seven plants growing along a horizon line. Hover a plant → soft caption. This is the analytics section, disguised as a metaphor.

5. **Campus Pulse**
   Image-rich anonymous feed, editorial magazine style. Three columns on desktop, one card per gratitude with a soft-toned photograph placeholder, quote pull, and heart. Feels like a Kinfolk spread, not a social feed.

6. **Karma Forest**
   Immersive full-bleed community visualization. An expanding SVG forest — each tree is a student's contribution this week. Jai's trees glow softly. Counter: "1,284 kind acts this week · 12 of yours." One line of copy, nothing else. Sits on its own dark editorial band.

7. **Quick Actions**
   Four large interactive tiles in a 2×2 grid: Breathe · Focus · Journal · Gratitude. Each tile is oversized, minimal, one icon, one word, one hover state. No progress rings, no stats — just doors.

8. **Weekly Reflection**
   AI-written prose summary of Jai's week. Serif, wide margins, reads like a page from a journal. One paragraph, hand-written feel. Signed "— Peace, on your behalf." Small "save to journal" action at the bottom.

## What Gets Removed

- Today's Composure ring + stat rail (folded into Hero + Mood Garden)
- Seven-day picker band (mood is now the Mood Garden metaphor)
- Sanctum triptych (Breath box, Pomodoro dial, Verse) → collapsed into Quick Actions
- Layered stillness deck → replaced by Today's Focus + ribbon
- Achievements grid → removed (streak already lives in sidebar)
- Ritual ribbon / The Hours sun-arc → removed
- Footer whisper / brand mark repetition → removed
- Emergency corner → single quiet link in sidebar footer only

## Design Principles

- **One section, one feeling.** No section shares a screen-height with another; each earns its own scroll.
- **Illustration over chart.** Mood Garden and Karma Forest are custom hand-drawn SVGs, not data viz.
- **Type does the work.** Fraunces for section titles at 56–72px, DM Sans for prose. Wide margins.
- **AI is present, not decorative.** Hero insight, Peace AI thread, and Weekly Reflection all pull real generated copy referencing Jai's data.
- **Palette stays.** Current soft-blue/lavender system — no palette change.

## Technical Notes

- Keep existing sidebar (glass tube, hover-expand, grouped nav, pinned footer). No changes.
- Mobile: same 8 sections stacked; illustrations scale down but never become icons.
- Peace AI section uses the existing `/api/chat` route; add session-level message history in component state.
- Mood Garden + Karma Forest: pure SVG components, no libraries.
- Today's Focus: uses one experience from the existing array; alternates ribbon shows the other 4.
- Weekly Reflection: on-mount `fetch('/api/chat')` with a summarization system prompt + week's stats.
